The WordPress Book Review Plugin enables you to add details such as title, author, cover photo and purchase links to each of your review posts.

== Description ==

Full documentation for the WordPress Book Review Plugin can be found on [Book Wookie](http://bookwookie.ca/wordpress-book-review-plugin/).

== Installation ==

1. Upload the `book-review` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Customize the settings by clicking on 'Book Review' in the 'Settings' menu.
